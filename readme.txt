Плагин позволяет выводить базу недвижимости на вашем сайте в режиме он-лайн. 


== Description ==

Полноценный поиск по общегородской базе недвижимости г.Санкт-Петербурга на вашем сайте. Модуль поддерживает виджеты. 


== Installation ==

1. Скопируйте файлы базы агентов в папку с плагинаями
2. Активируйте плагин
3. Настройте плагин в панели параметров
4. Если ваша тема поддерживает виджеты. Разместите виджеты в шаблоне, если виджеты не поддерживаются добавьте `<?php get_bazaagentov(); ?>` в любом месте вашего шаблона

== Advanced ==

Шаблон объекта вы можете задать самостоятельно в настройках плагина.

Доступные метатеги:
	%id% - номер объекта в общей базе
	%date% - дата публикации объекта
	%type% - тип объекта: квартира/комната и тд
	%adress% - адрес объекта
	%metro% - станция метро
	%metro_dist% - расстояние до метро
	%so% - общая площадь
	%sj% - жилая площадь
	%sk% - площадь кухни
	%sun_uzel% - тип санузла
	%telephone% - наличие телефона пишет + или -, %telephone_t% пишет "телефон" или ""
	%mebel%, %mebel_t% - наличие мебели
	%holod%, %holod_t% - наличие холодильника
	%washingmachine%, %washingmachine_t% - наличие стиральной машины
	%price% - стоимость в рублях
	%about% - описание объекта
	%komnat% - колличество сдаваемых комнат
	%komnat_all% - общее колличество комнат
	%photo% - фотография
	%floar% - этаж
	%floars% - этажей в здании

Работу плагина необходимо согласовать с администрацией bazaagentov.ru . Что бы адрес вашего сервера был внесен в белый список.
